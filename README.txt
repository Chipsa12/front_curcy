1. Разархивировать в любую папку
2. В консоли написать npm i и дождаться завершения установки пакетов
3. В консоли написать npm install -g typescript@3.9.7 и дождаться завершения установки пакета
4. В консоли написать npm start. Если после компиляции в консоли будет написано Compiled successfully значит всё ок
	и на http://localhost:8080 запустился проект. Проверить.
